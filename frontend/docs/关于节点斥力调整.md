节点间距调大
当前布局参数在第 57-67 行：

graph-container.tsxL57-L67

```
    layout: {
      type: 'd3-force',
      link: { distance: 180, strength: 0.6 },
      manyBody: { strength: -100 },
      collide: { radius: 40, strength: 1 },
      center: { strength: 0.05 },
      alphaMin: 0.002,
      alphaDecay: 0.03,
      velocityDecay: 0.5,
    },
```

d3-force 的斥力来自 三个参数的协同：

参数	当前值	作用	建议值
manyBody.strength	-80	负值 = 电荷斥力，绝对值越大排开越远	-200 ~ -300
link.distance	180	边连接的目标长度（等于是弹簧的静止长度）	300 ~ 400
collide.radius	40	节点碰撞检测半径，防止重叠	60 ~ 80
velocityDecay	0.5	速度衰减，降低一点让节点有更长稳定时间	0.4 ~ 0.3
