## 节点探索

1. 点击画布上的节点选中它，查看详情
2. 在右侧面板或详情卡片中点击 **"探索此节点"** 展开其邻居
3. 通过 **"每层加载数量"** 和 **"探索深度"** 滑块控制展开范围
4. 所有已加载的邻居加载完毕后，按钮会显示 **"已全部探索"**

## 高亮范围

左侧控制面板的滑块控制 BFS 高亮范围：

- **每层邻居上限**：每层最多显示多少个邻居节点
- **高亮深度**：从选中节点向外 BFS 多少层

## 业务域切换

上方工具栏的下拉框可以切换不同业务域，切换后图谱会重新加载。

## 导入/导出

- 点击工具栏的 **上传图标** 可以导入自定义 JSON 图谱数据
- 点击 **下载图标** 可以导出当前画布上的图谱数据
